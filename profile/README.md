# Sequel Ace — Portable Modern MySQL Client (Sequel Pro Fork)

## About Sequel Ace

Sequel Ace provides a complete MySQL and MariaDB database management environment for developers and database administrators. Sequel Ace combines table browsing, query editing, data manipulation, import/export, and SSH tunneling. Sequel Ace is a modern fork of Sequel Pro, updated for current macOS versions. Sequel Ace runs on desktop platforms. Sequel Ace is trusted by web developers, backend engineers, data analysts, and database administrators worldwide.

## Deep Dive — MySQL Client Notes

Table browsing: view table structure (columns, indexes, foreign keys), browse data
Query editor: syntax highlighting, autocomplete, multiple tabs, history
Data editing: add, edit, delete rows, inline editing, copy/paste rows
Import/export: CSV, SQL dump, XML, JSON, export query results
SSH tunneling: connect to remote MySQL via SSH, password or key authentication
